# Personal Portfolio Website

A dynamic personal portfolio website built using HTML, CSS, JavaScript, and Node.js.

## Features
- Personal profile section
- Skills and projects showcase
- Downloadable resume
- Contact form with Node.js backend
- MongoDB database for storing messages

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js
- MongoDB

## How to Run the Project

1. Install dependencies
npm install

2. Start the server
node server.js

3. Open in browser
http://localhost:3000
